# GoodKnightCommon

Common utilities library for the [GoodKnight](https://github.com/choiIsabelle/GoodKnight) chess AI project.

## Purpose

This repository provides shared functionality used across multiple GoodKnight components, including:
- FEN (Forsyth-Edwards Notation) to tensor conversion for neural network input

## Main Project

This is a supporting library for **GoodKnight**, a chess AI project. Visit the main repository at:
https://github.com/choiIsabelle/GoodKnight
